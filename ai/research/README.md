# Research Archive

This directory contains archived web research with citations and notes.

## Structure

Each research entry should include:
- **Link**: URL to the source
- **Summary**: Brief summary of findings
- **Citation**: Proper citation format
- **Why it matters**: Relevance to current work
- **Date**: When research was conducted

## Format

Create markdown files with the following structure:

```markdown
# Research: [Topic]

## Source
- **URL**: [link]
- **Date**: YYYY-MM-DD
- **Author**: [if applicable]

## Summary
[Brief summary of findings]

## Why It Matters
[Relevance to current work]

## Key Takeaways
- Point 1
- Point 2

## Related Files
- [Links to related research or implementations]
```

## Examples

- `cursor-hooks-research.md` - Research on Cursor IDE hooks
- `codex-cli-background-workers.md` - Codex CLI background terminal patterns
- `claude-code-subagents.md` - Claude Code subagent best practices
